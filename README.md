# The-Booksseeker
Созданию этого проекта поспособствовали новейшие IT тхнологии в лице искуственного интелекта.

Чтобы запустить сервис нужно открыть исполняемый файл "BooksSeeker.py"

Чтобы загрузить в сервис тексты книг, нужно нажать кнопку "загрузить файл" в интерфейсе программы

Демонстрация работы программы: https://disk.yandex.ru/i/YRLFqFCg15kKNQ
